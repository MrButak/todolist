# Todo list


### Technologies<br />
**Backend** - Node.js / Express.js<br />
**Frontend** - Vue.js

### To run

**1. In a terminal**
```
git clone https://github.com/MrButak/todolist.git`
cd todolist
npm i
npm start
```
**2. In a new terminal in the project root folder**
```
cd frontend
npm i
npm run dev
```
